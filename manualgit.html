MANUAL GIT


COMANDOS INICIAIS


GIT CONFIG –GLOBAL USER.NAME“SEU NOME”: este comando ajuda a configurar o nome de usuário que sera registrado nos seus commits


GIT CONFIG –GLOBAL USER. EMAIL “SEU EMAIL”: este comando configura o seu email que sera associado ao git


GERENCIAMETO DO SITE GIT


GIT INIT: Inicializa um repositório Git no diretório criando uma forma para rastrear e gerenciar alterações nos arquivos do seu site.


GIT ADD . OU GIT ADD "NOME DO ARQUIVO": este comando adiciona alteraçoes dos arquivos ou pastas deixando tudo pronto para um commit


GIT LOG: usando o git log nos conseguimos ver o histórico de commits já feitos

TRABALHO COM REPOSITORIOS REMOTOS


GIT REMOTE ADD ORIGIN "URL DO REPOSITORIO REMOTO" : ele adiciona um endereço ao git
